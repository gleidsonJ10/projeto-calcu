# projeto-calcu
#Este projeto contém uma calculadora simples feita em Bash Script e Python, desenvolvida para praticar comandos básicos no Linux e lógica de programação.
#Como executar o arquivo .sh
Dar permissão ao arquivo
 No terminal Linux digite:
 chmod 644 calculadora.sh
 chmod +x calculadora.sh
Executar o script
./calculadora.sh
O programa irá solicitar dois números e mostrar o resultado.

#Explicação do código Python

O código Python funciona da seguinte forma:

Utiliza input() para receber números do usuário

Converte os valores para número usando int() ou float()

Realiza operações matemáticas

Mostra os resultados utilizando print()
Exemplo básico:
 n1 = int(input("Digite um número: "))
n2 = int(input("Digite outro número: "))

soma = n1 + n2

print("Resultado:", soma)
